## ğŸ’¡ Pro Tips

1. **First Run Takes Time**: PDF extraction and embedding generation take ~10-15 minutes on first run. This is normal.

2. **CPU is Fine**: Everything runs on CPU by default. GPU optional for speed.

3. **Similarity Scores**: Typically 0.5-0.8. Use these thresholds:
   - `> 0.8` - Excellent match
   - `0.6-0.8` - Good match
   - `0.4-0.6` - Possible match
   - `< 0.4` - Weak match

4. **Scaling**: Can handle 100,000+ documents efficiently.

5. **Customization**: Edit `requirements_chroma.txt` to use different embedding models.

**Solution**: Normal for first run. Be patient.
